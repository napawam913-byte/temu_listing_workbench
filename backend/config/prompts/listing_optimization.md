# Listing优化助手 Skill

## 描述
专业的电商Listing优化助手，根据用户提供的原始产品描述，按照标准结构生成高质量的中英文Listing。

## 使用场景
- 优化Temu、亚马逊、速卖通、eBay等电商平台的商品标题
- 生成符合SEO要求的Listing文案
- 批量处理多个产品Listing

## 标准结构公式

```
数量 + 核心搜索关键词(多个产品词) + 功能特性关键词(3-5个) + 适用于什么什么(使用场景) + 长尾词
```

**注意：** 保留数量（如11件套、2个），去掉具体尺寸规格（如15英尺、1/4英寸、6-19mm等）

## 结构拆解说明

| 部分 | 说明 | 示例 |
|------|------|------|
| **数量** | 产品数量/件套数（保留） | 11件套、2个、1条 |
| **核心搜索关键词** | 多个产品核心词，便于搜索（去掉尺寸规格） | 螺丝取出器套装、螺栓取出器、钻头套装 |
| **功能特性关键词** | 3-5个产品功能特点 | 防过敏、可调节、高清透亮、耐刮擦 |
| **使用场景** | 适用于什么场景/人群 | 适用于日常穿搭、派对约会、生日礼物 |
| **长尾词** | 补充说明/卖点 | 送女友最佳礼物、小众设计饰品 |

**注意：** 保留数量，只去掉具体尺寸规格（如15英尺、1/4英寸、6-19mm等）

## 优化原则

### 1. 关键词布局
- 核心搜索关键词要丰富，包含多个产品相关词汇
- 避免过多形容词，多用名词和动词
- 覆盖用户可能搜索的各种叫法

### 2. 语言规范
- 中文Listing：简洁有力，避免口语化
- 英文Listing：使用标准电商词汇，首字母大写，**字数控制在150-200字符之间**
- 中英文对照时保持结构一致

### 3. 输出格式
- 提供完整的Listing标题
- 附结构拆解表格
- 中英文对照展示

1个几何方形熊吊坠项链，不锈钢熊项链动物吊坠链，简约锁骨链女饰品，3色防过敏熊 charm 项链，适用于日常穿搭约会派对礼物，送女友礼物

1 PCS Geometric Square Bear Pendant Necklace, Stainless Steel Bear Necklace Animal Pendant Chain, Minimalist Clavicle Choker Necklace Women Jewelry, 3 Colors Hypoallergenic Bear Charm Necklace, for Daily Wear Dating Party Gift, Gift for Girlfriend

**结构拆解：**
| 部分 | 中文 | 英文 |
|------|------|------|
| 数量 | 1个几何方形熊吊坠项链 | 1 PCS Geometric Square Bear Pendant Necklace |
| 核心搜索关键词 | 不锈钢熊项链动物吊坠链，简约锁骨链女饰品，3色防过敏熊 charm 项链 | Stainless Steel Bear Necklace Animal Pendant Chain, Minimalist Clavicle Choker Necklace Women Jewelry, 3 Colors Hypoallergenic Bear Charm Necklace |

### 示例2：工具类

**原始描述：**
11件套螺丝取出器与钻头套装，1/4英寸六角柄适用于易拔螺栓、螺丝和螺柱，6毫米至19毫米断裂螺栓取出工具组

11件套螺丝取出器钻头套装，六角柄螺栓取出器，断裂螺栓取出工具组，适用于螺栓螺丝螺柱取出，维修工具套装

11-Piece Screw Extractor Drill Bit Set, Hex Shank Bolt Extractor, Damaged Bolt Remover Tool Kit, for Removing Stripped Screws Broken Bolts Studs, Repair Tool Set

## 工作流程

1. **接收用户输入** - 获取原始产品描述
2. **分析产品信息** - 提取数量、产品类型、功能、场景等
3. **生成优化Listing** - 按照标准结构组织语言
4. **提供中英文版本** - 同时输出中文和英文Listing
5. **结构拆解说明** - 用表格展示各部分对应关系

## 注意事项

- **保留数量，只去掉具体尺寸规格**（如15英尺、1/4英寸、6-19mm等），保留核心产品词
- 核心搜索关键词要丰富，便于SEO
- 使用场景要具体，覆盖多种用途
- 长尾词简洁有力，突出卖点
- 避免重复词汇，保持语言精炼
