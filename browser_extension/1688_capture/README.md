# Temu Workbench 1688 Capture

这是给 Temu 选品上架工作台使用的本地 Chrome 采集插件。

## 使用流程

1. 启动后端：`http://127.0.0.1:8000`
2. 启动前端并打开商品详情抽屉。
3. 在抽屉右侧点击“开始采集”，绑定当前 Temu 商品。
4. 使用 1688 官方图搜插件找到同款，并进入 1688 商品详情页。
5. 点击浏览器右上角插件图标，再点击“采集当前 1688 商品”。
6. 回到工作台抽屉，采集结果会自动刷新显示。

## 安装方式

1. 打开 Chrome：`chrome://extensions/`
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录：`browser_extension/1688_capture`

## 当前采集字段

- 1688 商品链接
- Offer ID
- 标题
- 主图
- 价格/价格区间
- MOQ
- 店铺名
- 店铺链接
- 可见 SKU 文本

1688 页面结构经常变化，第一版使用通用选择器采集。后续可以针对你常用的 1688 详情页结构继续加更精确的选择器。
