# 1688 商品录入 Chrome 插件

这是把原来的桌面版商品录入工具改成 Chrome 插件后的版本，目录本身就是一个可直接加载的插件文件夹。
当前版本使用 Chrome 原生侧边栏模式，点击扩展图标后会在浏览器右侧打开工具界面。打开后不要关闭侧边栏，切换 1688 商品页会自动刷新当前页数据。

## 功能

- 在 1688 商品详情页自动抓取：
  - 当前页面价格
  - 首个重量
  - 当前 URL
- 保留原来的利润计算逻辑
- 支持在插件页面里直接 `Ctrl+V` 粘贴图片
- 审核时先添加到本地列表，最后一次性导出 Excel
- 支持倍数/份数参数，例如同一个商品买 3 份
- 保存时自动生成：
  - `records.xlsx`

## 目录说明

- `manifest.json`：Chrome MV3 插件清单
- `popup.html` / `popup.css` / `popup.js`：插件主界面
- `content.js`：1688 页面抓取脚本
- `config.json`：利润计算配置
- `excel_writer.js`：纯前端 Excel 导出逻辑
- `storage.js`：目录选择和本地文件读写

## 如何加载

1. 打开 Chrome。
2. 进入 `chrome://extensions/`
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择当前这个文件夹：

`C:\Users\Administrator\Desktop\计算器源码\chrome_1688_recorder`

## 如何打开侧边栏

1. 打开一个 1688 商品详情页。
2. 点击 Chrome 工具栏里的这个扩展图标。
3. 插件会在浏览器右侧以侧边栏形式打开。
4. 保持这个侧边栏打开，之后换商品页不需要重新点击扩展。

## 如何使用

1. 先打开一个 1688 商品详情页。
2. 点击插件图标打开右侧边栏。
3. 插件会自动抓取当前页的单件价格、重量和 URL。
4. `单件` = 1688 当前货值 + 页面下方运费。
4. 手动补充 `NAME`、`SKC`、`核价` 等字段。
5. 在插件页面里直接粘贴商品图片。
6. 每审核完一条，点击“添加到列表”。
7. 全部审核完后，点击“导出 Excel”。
8. 在浏览器弹出的保存窗口里选择 `records.xlsx` 的保存位置。

## 配置文件

插件会直接读取同目录下的 `config.json`，以后你只要改这个文件就行，不需要改代码。

当前默认配置：

```json
{
  "first_leg_surcharge": 7.5,
  "first_leg_price_per_kg": 80,
  "last_leg_fee": 23,
  "last_leg_subsidy": 21,
  "after_sales_rate": 0.05,
  "discount_rate": 1.0
}
```

## 说明

- 这个版本是纯前端插件实现，没有再依赖 Python。
- 为了避免 Edge/侧边栏里选择目录导致浏览器崩溃，当前版本不再使用 `showDirectoryPicker()`。
- 记录会保存在插件本地数据库中，点击“添加到列表”不会弹保存窗口。
- 点击“导出 Excel”时才会重新生成完整 `records.xlsx` 并由浏览器下载保存。
- 导出成功后会自动清空本地列表，下一批不会重复带上旧记录。
- 推荐使用 Chrome 测试。Edge 虽然也是 Chromium 内核，但原生侧边栏和目录授权策略可能与 Chrome 有差异；如果 Edge 里侧边栏不能保持常驻，请优先用 Chrome 加载这个文件夹测试。
