# Temu Workbench 1688 Capture

这是给 Temu 选品上架工作台使用的本地 Chrome 采集插件。

## 使用流程

1. 启动后端：`http://127.0.0.1:8000`
2. 进入任意 1688 商品详情页。
3. 点击浏览器右上角插件图标，打开右侧采集侧边栏。
4. 侧边栏会自动预抓当前 1688 页面，优先抓取价格、运费、含运费单件和重量。
5. 确认数据后，点击“加入采集列表”。
6. 在侧边栏“采集列表”中勾选一个或多个商品，再选择：
   - “加入选中为货源素材”：加入当前监听商品的 1688 货源素材列表。
   - “加入选中到商品列表”：作为新的 1688 商品加入前端商品列表。
7. 加入商品列表成功后，插件会自动刷新已打开的前端工作台页面。
8. 采集列表支持单个商品直接加入商品列表、加入货源素材或删除。

## 安装方式

1. 打开 Chrome：`chrome://extensions/`
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录：`browser_extension/1688_capture`

## 当前采集字段

- 1688 商品链接
- Offer ID
- 标题
- 主图
- 价格/价格区间
- MOQ
- 店铺名
- 店铺链接
- 可见 SKU 文本
- 运费
- 含运费单件成本
- 重量 kg

1688 页面结构经常变化，当前版本结合通用选择器和页面内联脚本采集。价格、运费、重量会进行多次延迟重试；SKU 目前仍以可见规格文本为主，后续可以继续针对常用详情页结构增强 SKU 价格、库存和图片。
